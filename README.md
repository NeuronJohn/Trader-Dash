# Topstep Discipline Trader Dashboard (Vercel)

This is a Vite + React + Tailwind project wired for Vercel deployment.

## Local run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

Vercel auto-detects Vite projects. Import this folder/repo into Vercel or deploy from the CLI:

```bash
npm i -g vercel
vercel --prod
```

## Main app file

- `src/App.jsx`

## Notes

- Data is saved in browser localStorage.
- Export/import creates or reads a JSON backup.
